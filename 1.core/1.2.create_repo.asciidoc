Git basics
----------

- git init
- the .git folder
- git log
- git status
- git add
- git commit
- git diff

To initialise a git repository, run 'git init' from within the root folder
of the source you want to manage.

----
git init
----

This initialises a database

----
cd .git
ls
# branches
# COMMIT_EDITMSG
# config
# description
# HEAD
# hooks
# index
# info
# logs
# objects
# refs
----


call out HEAD and refs - master is the default, just convention
